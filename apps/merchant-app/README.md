
# apps/merchant-app/README.md

## Merchant App

The **Merchant App** is the dashboard for merchants using the Paytm Clone platform. It provides tools to manage payments, view balances, and monitor transactions.

### Development

```bash
pnpm dev --filter merchant-app
```

Open: [http://localhost:3000](http://localhost:3000)

### Docker

```bash
docker compose up merchant-app
```
