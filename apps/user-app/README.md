# apps/user-app/README.md

## User App

The **User App** is the customer-facing application of the Paytm Clone. It allows users to interact with the platform, manage balances, and initiate transactions.

### Development

```bash
pnpm dev --filter user-app
```

Open: [http://localhost:3001](http://localhost:3001)

### Docker

```bash
docker compose up user-app
```
